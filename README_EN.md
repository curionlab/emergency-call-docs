# Emergency Call System - Documentation

This repository provides comprehensive documentation for the Emergency Call System.

## System Overview

An emergency video/voice call system combining Web Push (VAPID) and WebRTC (PeerJS).  
Receivers receive notifications and calls automatically start when they tap the notification.

### Repositories

- **Client**: https://github.com/curionlab/emergency-call-client
- **Server**: https://github.com/curionlab/emergency-call-server
- **Docs (this repository)**: https://github.com/curionlab/emergency-call-docs
